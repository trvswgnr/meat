# Meat
Meat is a no-nonsense library for Bash.

Current in very early alpha. Use with discretion.

### Installation
Clone this repo and add `/meat` to your $PATH in .bash_profile, .bashrc, etc...

```bash
export PATH="/path/to/meat:$PATH"
```
